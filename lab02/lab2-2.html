HTML

하이퍼 텍스트 마크업 언어(Hyper Text Markup Language, HTML)는 웹 페이지 표시를 위해 개발된 지배적인 마크업 언어다.

CSS

캐스케이딩 스타일 시트(Cascading Style Sheet)는 마크업 언어가 실제 표시되는 방법을 기술하는 스타일 언어다.

마크업 언어(ex: HTML)가 웹사이트의 몸체를 담당한다면 CSS는 옷과 액세서리처럼 꾸미는 역할을 담당한다.

현재 개발 중인 CSS3의 경우 다양한 기능이 추가되었다.

JavaScript

자바스크립트(JavaScript)는 객체 기반의 스크립트 프로그래밍 언어이다.
